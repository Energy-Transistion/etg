etg.simulation.agent module¶

All classes and methods having to do with agents

class etg.simulation.agent.Agent(simulation, income, ambition, certainty, need_money, need_green, need_safety, need_government_money, energy_consumed)[source]¶

Bases: etg.simulation.entity.Entity

A class to represent a single agent in the artificial population. It contains information about the state of the agent (income, certainness, friends, etc) and methods for updating the agent on each time step. The agents are created when the simulation is started, and do not have to be made manually.

use_comparison()[source]¶

If the agent is unsatisfied and the agent is uncertain, then it will find the company that most of its friends are clients of.

use_deliberation()[source]¶

If the agent is certain about itself, but unsatisfied, it will find the best company and join that.

use_imitation()[source]¶

If the agent is satisfied, but uncertain, it will pick the most common company among its friends.

use_repetition()[source]¶

If the agent is satisfied and certain, it will just stay with its current provider.
