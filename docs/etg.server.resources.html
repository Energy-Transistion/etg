A function to create a resource that allows access to the javascript files. Takes the path to a directory with javascript files.

get_static_html(templates, text_dir, **kwargs)[source]¶

A function to create a resource that allows access to the static html files. Takes the path to a template and a YAML file with values that should go into this template. The files in the returned resource have the same name as the files in text_dir, but with extension ‘.html’.

Parameters:
  • template – The template that the text should be rendered in. It can take any number of arguments.
  • text_dir – A folder of YAML files that give the information that the template needs.
get_styles_resource(style_dir)[source]¶

A function to create a resource that allows acces to the styles. Takes the path to a directory with style files as either scss or css.

etg.server.resources.render_file(file, templates, fall_back_template, **kwargs)[source]¶

Render a given file-like object in the corresponding template. All the extra options are passed on to the template as well.

Parameters:
  • file – The file to render.
  • templates – The Jinja2 environment to get the template from.
  • fall_back_template – The name of the template to use if no template is specified in the file.
