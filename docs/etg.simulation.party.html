etg.simulation.party module¶

All the classes and methods having to do with the political parties in the game.

class etg.simulation.party.Party(simulation, config, name)[source]¶

Bases: etg.simulation.entity.Entity

A class to represent the parties in the simulation. It contains information about the state of the party (funds, policies, voters, etc) and methods for updating the party and validating user input. The parties are created when a player joins, and should not be created manually.

campaign()[source]¶

The party can campaign in order to get more voters to vote for them. It then does need the required amount of money before it can start.

campaign_reach¶

What percentage of the population is affected by the campaigns of this party.

greenness¶

The greenness of the policies of this party.

percentage_voters¶

The percentage of the population that would vote for this party in the last poll, as a percentage.

receive_donation(amount)[source]¶

The party received a donation.

safety¶

The safeness of the policies of this party.

tick()[source]¶

In a turn, the party campaigns to get more voters.

voters¶

All the voters for this party, as an AgentSet.
