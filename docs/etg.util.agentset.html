etg.util.agentset module¶

A class to mimic Netlogos agentsets iteration behaviour

class etg.util.agentset.AgentSet(*args, _list=None, _iter=None)[source]¶

Bases: object

A class to mimic Netlogo’s agentsets random iteration behaviour. Also has some convience methods that making common sequence operations for which order does not matter faster.

An object can be initialized by giving the agents that shoudl initially be in the set in its constructor.

add_agent(agent)[source]¶

Add an agent to the agent set. If the agent is already in the set, do nothing.

filter(predicate)[source]¶

A more efficient filter that works directly on the underlaying datastructure, and therefore does not shuffle the list first. If order is not important for the filtering, this method is preferred over the built in filter

n_of(number)[source]¶

Return number of agents from the set, as an AgentSet.

one_of()[source]¶

Return a random agent from the set.

remove_agent(agent)[source]¶

Remove an agent from the set. If the agent is not in the set, do nothing.
